## 网站概述

这是一个为星际酒馆PVP模式设计的**猜牌辅助工具**。玩家通过不断进场卡牌并记录是否"获得隐刀"（接近反馈），逐步缩小对手预言牌（隐藏卡牌）的可能范围，类似于一个逻辑推理/排除法工具。

## 泽拉图技能描述

```
角色技能: 探寻答案
游戏开始时，核心包的随机一张卡牌成为预言牌。任意卡牌进场时，若满足以下条件中的一个，则获得1名黑暗圣堂武士(精英)：
- 种族与预言牌种族相同
- 卡牌价值与预言牌价值相差小于或等于200
- 初始单位数量与预言牌相同
每回合，可使用技能猜测一次预言牌：若猜中，则揭示预言牌并将奖励的黑暗圣堂武士(精英)数量提升至2，然后生成新的预言牌
此外，若前3次就猜中，那么在上述符合条件的卡牌进场时，还能额外获得1次免费刷新
```

## 主要功能

### 1. 扩展包数据集管理（左侧栏）

- 左侧栏以竖向复选框列表展示扩展包开关，包括：核心（默认启用且不可关闭）、军备竞赛、作战计划、卷土重来、时不我待、重装上阵、穷兵黩武、一念之差、身经百战、比特狂潮、同卵双狗
- 勾选/取消扩展包时，通过动态 `<script>` 标签异步加载卡牌数据（`window._packData` 全局注册表）
- 最多勾选两个额外扩展包（核心包不计入限制）
- 切换扩展包时自动清空猜测记录
- 显示已加载的总卡牌数

### 2. 卡牌选择与筛选（中央面板）

- **种族筛选按钮行**: 可按种族（星灵/异虫/人族/中立）筛选卡牌
- **等级筛选按钮行**: 根据当前种族筛选结果，动态显示可用等级按钮
- **卡牌快捷按钮行**: 当同时选中种族和等级后，以按钮形式直接展示所有匹配卡牌，点击即选中
- **已用卡牌标记**: 已进场过的卡牌按钮会置灰并加删除线，不可重复选择

### 3. 进场猜测与反馈记录

- 选择一张卡牌后，通过切换按钮标记反馈：**"奇数"**（close）或 **"偶数/0"**（not_close）
- 点击"进场！"按钮提交一次猜测（有点击闪光动画反馈）
- 判断"接近"（close）的规则为：进场卡牌与预言牌 **种族相同** 或 **单位数相同** 或 **价值差值 ≤ 200**
- 相同卡牌不可重复进场

### 4. 进场卡牌记录（中央面板 - 历史表格）

- 以可滚动表格形式展示所有已提交的猜测记录
- 每行显示：序号、卡牌ID、种族、等级、单位数、价值、是否获得隐刀
- 每行提供"移除"按钮，可删除单条记录
- 顶部提供"清除全部"按钮（带确认弹窗）

### 5. 可能的预言牌推算（右侧栏 - 核心逻辑）

- 根据所有已提交的猜测记录，用排除法计算出**仍然可能的预言牌**列表
- 只从核心卡牌集（isCoreSet）中筛选候选牌
- 对每张候选卡牌验证其是否与所有猜测记录一致
- 显示剩余候选牌数量占总核心卡牌数的比例（如 "15/120"）

#### 约束推断摘要

- 基于"获得隐刀"的记录，自动推断预言牌在三个维度（种族、单位数、价值）的约束状态
- 使用**约束传播算法**：当某张"获得隐刀"的卡牌只剩一个可能的匹配维度时，锁定该维度
- 约束状态分三级：**已锁定**（确定值）、**已缩小**（排除部分可能）、**未知**
- 以彩色标签形式直观展示各维度约束

#### 匹配维度与区分度

- 每张候选牌显示**匹配维度标签**（种族/单位数/价值），标识其通过哪些维度与已进场的"获得隐刀"卡牌匹配
- **区分度评分**（0-5分，5格方块可视化）：综合稀缺度加权评估每张候选牌的可能性
- 支持四种排序方式：**区分度**（默认）、按种族、按单位数、按价值
- 与已猜测卡牌接近的候选牌行会有高亮背景

#### 预言牌确认

- 每张候选牌提供"预言牌"按钮，点击后弹出模态确认框
- 确认后清空猜测记录，开启下一轮预言的猜测

## UI/交互设计特点

- **SC2 神族暗色主题**: 深空底色（#040e2f）配神族青色强调色（#00bcd4），金色（#c8aa6e）高亮、预言紫（#9c5fff）作为辅助色
- **背景图**: 固定背景图片 + 面板纹理背景，增强星际氛围
- **三栏布局**: 左侧栏（卡包选择）| 中央区（进场操作 + 历史记录）| 右侧栏（候选推算）
- **定制化边框**: 侧栏使用 `border-image` 实现 SC2 风格边框装饰
- **SC2 风格滚动条**: 自定义滚动条外观，深色轨道 + 渐变拖块
- **固定顶部标题栏**: Header 固定于页面顶部，含 Logo 图标与标题
- **模态确认框**: 预言牌确认弹窗带滑入动画和模糊背景遮罩
- **响应式布局**: CSS 媒体查询适配平板（768px）和移动端（480px），小屏幕下三栏变单栏
- **交互动画**: 按钮点击闪光、模态框滑入、hover 发光效果
- **问号提示气泡**: 标题旁的 `?` 图标，hover/focus 时显示使用说明
